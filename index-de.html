FEATURES

Valkyrie wurde für Benutzer geschaffen, die ihre eigenen Abenteuer kreieren und sie mit der Community teilen möchten.
  • Erstellen
  • Spielen

Erstellen deine eigene Quest

Sei der Game Master und erstelle deine eigenen Geschichten.

Scenario Editor

Verwenden den Szenario-Editor, um Tokens und Spielplanteile zu platzieren.

Event Driven

Erstelle events for Tokens, Spielplanteile, Helden und mehr.

Teile Inhalte mit der Community

Teste dein Scenario

Leave a Feedback

Kurzes Einführungsvideo

Windows

Führen Sie das Installationsprogramm aus, installieren Sie es am gewünschten Ort und starten Sie dann Valkyrie. Der Import kann eine Minute dauern, seien Sie also nicht zu voreilig. Falls der Import nicht abgeschlossen wird oder nicht funktioniert, lesen Sie bitte den Abschnitt Troubleshooting.

Erfordert die Installation der FGG-Anwendung für das Brettspiel, das du spielen möchtest:

  • Mansions of Madness Second Edition erfordert die Villen des Wahnsinns app.
  • Descent: Die Reise ins Dunkel (Zweite Edition) erfordert die Descent: Weg zum Ruhm app.
    • Neuste Version als .zip herunterladen Neuste Version als .exe herunterladen

Android

Wenn Sie bereits eine Version von Valkyrie installiert haben, deinstallieren Sie diese bitte zuerst.

Installieren Sie die offizielle App über den Google Play Store:

  • Villen des Wahnsinns (Zweite Edition)
  • Descent: Wege zum Ruhm

1. Starten Sie die offizielle App und warten Sie, bis der Ladevorgang abgeschlossen ist (der Warte-Cursor im Hauptmenü ist verschwunden).

2. Aktivieren Sie "Unbekannte Quellen" unter Einstellungen → Sicherheit → Unbekannte Quellen. Kopieren Sie die Valkyrie APK-Datei auf Ihr Gerät.

3. Führen Sie die Valkyrie APK zur Installation aus. Nach der Installation finden Sie eine Anwendung namens "Valkyrie" in Ihrer Liste der Anwendungen.

4. Ausführen. Lassen Sie alle Berechtigungen zu, die die Anwendung anfordert. Klicken Sie auf die Schaltfläche "Importieren" in der Valkyrie-Anwendung. Der Import kann eine Weile dauern. Nach dem Import sind Sie in der Lage, den gewünschten Spieltyp auszuführen.

Wenn Sie einen schwarzen Bildschirm erhalten, gehen Sie bitte zu den App-Einstellungen und überprüfen Sie, ob der Zugriff auf den Systemspeicher eingeschaltet ist.

Wenn der Import fehlschlägt, stellen Sie sicher, dass Sie genügend internen Speicherplatz zur Verfügung haben. Jeder Spieltyp erfordert bis zu 300 MB. Um besonders sicher zu gehen, sollten Sie 1 GB freien Speicherplatz zur Verfügung haben.

MAC OS

Mac OS Gatekeeper muss deaktiviert sein:

Suchen Sie im Finder die App, die Sie öffnen möchten. Verwenden Sie dazu nicht Launchpad. Mit Launchpad können Sie nicht auf das Kontextmenü zugreifen.

1. Drücken Sie die Strg-Taste, klicken Sie auf das Anwendungssymbol und wählen Sie dann Öffnen aus dem Kontextmenü. Klicken Sie auf Öffnen..

2. Die App wird als Ausnahme zu Ihren Sicherheitseinstellungen gespeichert, und Sie können sie in Zukunft wie jede andere registrierte App durch einen Doppelklick öffnen.

Falls der Import nicht abgeschlossen werden kann, versuchen Sie bitte, Walküre mit erhöhten Privilegien auszuführen.

Stellen Sie außerdem sicher, dass Sie Ordner-Berechtigungen haben. Verschieben Sie Ihren Valkyrie-Download in den Ordner Programme innerhalb der Finder-Anwendung. Führen Sie dann chmod +x /Applications/Valkyrie.app/Inhalte/MacOS/Valkyrie in der Terminal-Anwendung aus.

Linux

Auch für Linux gibt es keinen automatischen Import:

1. Inhalte kopieren/<tippen sie>/ffg von einem anderen importierten Speicherort

2. Führen sie -import <Speicherort aus>, wobei <Speicherort> der Ort ist, an dem Ihre Wege zum Ruhm oder Villen des Wahnsinns Dateien liegen.

Zum Beispiel /home/ <Benutzername< /.local/share/Steam/steamapps/common/Descent Road to Legend für Wege zum Ruhm.

Fire OS

Benutzen Sie die Android-Version von Valkyrie, es wird auch unter Fire OS funktionieren.

Sie müssen die offizielle OBB-Datei manuell herunterladen und installieren (Sie erhalten sie hier)

Kopieren Sie die Datei nach <storage>/Android/obb/<game> (Wenn der Pfad nicht existiert, erstellen Sie ihn)

1. <Speicher> sollte der "Interne Speicher" sein. /storage/emulated/0 in den meisten Fällen.

<Spiel> muss entweder sein:

Villen des Wahnsinns: com.fantasyflightgames.mom/ main.906.com.fantasyflightgames.mom.obb

Wege zum Ruhm: com.fantasyflightgames.rtl/ main.459.com.fantasyflightgames.rtl.obb

Aktivieren Sie "Unbekannte Quellen" unter Einstellungen → Sicherheit → Unbekannte Quellen. Kopieren Sie die valkyrie APK-Datei auf Ihr Gerät. Führen Sie das Valkyrie APK zur Installation aus. Nach der Installation finden Sie eine Anwendung namens "Valkyrie" in Ihrer Liste der Anwendungen. Führen Sie sie aus. Lassen Sie alle Berechtigungen zu, die die Anwendung anfordert. Klicken Sie auf die Schaltfläche "Importieren" innerhalb der Walküre-Anwendung. Der Import kann eine Weile dauern. Nach dem Import sind Sie in der Lage, den gewünschten Spieltyp auszuführen.

iOS

Obwohl wir aufgrund der von Apple erzwungenen App Store-Anforderungen keinen Client ausliefern, können Sie Valkyrie mit KinoConsole von Ihrem Windows-PC aus auf iOS streamen. KinoConsole.

Installieren Sie einfach Valkyrie und KinoConsole Server auf Ihrem PC (starten Sie Ihren PC bei Bedarf neu). Starten Sie die neue, installierte Anwendung "Kinoni Streamer". Gehen Sie auf "Schnellstart", "Durchsuchen..." und fügen Sie die Valkyrie.exe hinzu (normalerweise unter C:\Programmdateien (x86)\Valkyrie\Valkyrie.exe zu finden).

Installieren sie die App KinoConsole aus dem App Store. Wenn sich Ihr iOS-Gerät und der PC im selben WLAN befinden, sollte der Server automatisch erkannt werden und Sie finden "Valkyrie" in der Liste der startbaren Apps. Siehe auch: Offizielle Setup-Anleitung.

Ältere iOS-Versionen (< iOS 10) unterstützen KinoKonsole nicht. Sie können versuchen, Splashtop Personal auf diesen zu verwenden
