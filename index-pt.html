Windows

Rode o instalador e instale na pasta de sua preferência, e então, inicie o Valkyrie.

Você pode importar os dados usando um dos dois métodos abaixo:

  • Importar do aplicativo oficial: Permite que você importe diretamente dos aplicativos oficiais Mansions of Madness ou Descent: Road to Legend.
  • Importar de ZIP: Permite importar de um arquivo zip baixado. O arquivo zip deve ser baixado das URLs listadas no ZIP Import Guide para o seu sistema operacional.

Android

Aviso: Actualmente a Valkyrie só é suportada pelo Android 10 ou inferior. Ver troubleshooting para detalhes. << /p>

Se você já tem uma versão do Valkyrie instalada, por favor desinstale primeiro.

Nota: A importação de aplicativos oficiais não é possível devido a restrições de acesso do Android.

1. Habilite "Unknown Sources" (Fontes desconhecidas) em Settings → Security → Unknown Sources (Configurações → Segurança → Fontes Desconhecidas). Copie o APK do Valkyrie para seu dispositivo.

2. Rode o APK do Valkyrie para instalar. Após instalar você vai encontrar um App chamado "Valkyrie" entre seus aplicativos.

3. Rode. Permita todas as permissões que o App peça.

4. Use o botão Importar ZIP para importar conteúdo. O arquivo zip deve ser baixado das URLs listadas no ZIP Import Guide.

5. Em seguida, verifique o Manual de instalação do Android.

MAC OS

Mac OS Gatekeeper deve estar desabilitado:

No Finder, localize o app que você quer abrir. Não use o Launchpad para fazer isso. Launchpad não deixa você acessar o menu de atalhos.

1. Pressione a tecla Control e clique no ícone do app, e então abra a partir do menu de atalho. Clique Abrir.

2. O app está salvo como uma exceção em suas configurações de segurança, e você pode abrir no futuro clicando duas vezes, como qualquer outro aplicativo.

Você pode importar os dados usando um dos dois métodos abaixo:

  • Importar do aplicativo oficial: Permite que você importe diretamente dos aplicativos oficiais.
  • Importar de ZIP: Permite importar de um arquivo zip baixado. O arquivo zip deve ser baixado das URLs listadas no ZIP Import Guide.

Além disso, garanta que você não tem nenhuma permissão na pasta. Mova seu download fo Valkyrie para a pasta Applications dentro do Finder. Então execute no terminal "chmod +x /Applications/Valkyrie.app/Contents/MacOS/Valkyrie"

Linux

Para Linux não há importação automática:

  • Você pode rodar com -import <location> onde <location> são os dados de Road to Legend ou Mansions of Madness. Por exemplo /home/ <username> /.local/share/Steam/steamapps/common/Descent Road to Legend para Road to Legend.
  • Alternativamente você pode usar Importar de ZIP: Permite importar de um arquivo zip baixado. O arquivo zip deve ser baixado das URLs listadas no ZIP Import Guide.
