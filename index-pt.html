Valkyrie é um App para que usuários criem suas aventuras/cenários para os jogos de tabuleiro da Fantasy Flight Board Games e compartilhar com jogadores no mundo todo.

WINDOWS ANDROID MAC OS LINUX iOS

Valkyrie foi criado para que usuários possam criar suas próprias aventuras e compartilhar com a comunidade.

Windows

Rode o instalador e instale na pasta de sua preferência, e então, inicie o Valkyrie. A importação pode levar alguns minutes, então não fique ansioso. Se a importação não terminar ou não funcionar verifique a seção de Troubleshooting (em inglês).

É necessário instalar o App da FFG correspondente ao boardgame que você quer jogar:

  • Mansions of Madness Second Edition precisa do Mansions of Madness app.
  • Descent: Journeys in the Dark (Second Edition) precisa do Descent: Road to Legend app.

Android

If you already have a version of Valkyrie installed, please uninstall it first.

1. Enable "Unknown Sources" under Settings → Security → Unknown Sources.

2. Click "Download Último" button below to download the Valkyrie .apk file to your device or copy it from another device.

3. Run the valkyrie .apk file to install Valkyrie. After the installation finished, you will find an app named "Valkyrie" in your list of apps.

4. Run the app.

5. Allow all permissions the app requests.

6. For all further steps including data import from the official apps please see Android installation manual.

MAC OS

Mac OS Gatekeeper deve estar desabilitado:

No Finder, localize o app que você quer abrir. Não use o Launchpad para fazer isso. Launchpad não deixa você acessar o menu de atalhos.

1. Pressione a tecla Control e clique no ícone do app, e então abra a partir do menu de atalho. Clique Abrir.

2. O app está salvo como uma exceção em suas configurações de segurança, e você pode abrir no futuro clicando duas vezes, como qualquer outro aplicativo.

Se a importação não for completa tente rodar Valkyrie como mais permissões.

Além disso, garanta que você não tem nenhuma permissão na pasta. Mova seu download fo Valkyrie para a pasta Applications dentro do Finder. Então execute no terminal "chmod +x /Applications/Valkyrie.app/Contents/MacOS/Valkyrie"

Linux

Para o Linux também não há importação automática:

1. Copie content/<type>/ffg para outra instalação or

2. Rode com -import <location> where <location> é Road to Legend ou Mansions of Madness data. Exemplo /home/<username> /.local/share/Steam/steamapps/common/Descent Road to Legend for Road to Legend.

iOS

Embora não enviemos um cliente devido aos requisitos forçados da App Store da Apple, podes transmitir o Valkyrie para iOS a partir do teu computador usando Steam Link.

Basta instalar o Valkyrie e o Steam no seu PC. Inicie o aplicativo Steam no seu computador e o aplicativo de link de vapor no seu dispositivo IOS. Depois disso, inicie o Valkyrie no seu computador para transmiti-lo.
